# Oh-My-Codex 分享版 HTML PPT

## 打开方式

- 直接打开 `index.html`。
- 或使用任意静态文件服务访问本目录。

## 演示控制

- `←` / `→`、`PageUp` / `PageDown`：前后翻页
- `Space`：下一步或下一页
- `Home` / `End`：首尾页
- `Esc`：打开或关闭 15 页索引
- 鼠标滚轮、触屏左右滑动也可翻页

第 7 页 Pipeline 会逐步点亮五个阶段，全部点亮后才进入下一页。

## 文件

- `index.html`：15 页演示入口
- `assets/`：本地图片与离线动效依赖
- `output/deck-preview.jpg`：全页总览
- `大纲-v1.md`：内容结构与主题节奏记录

内容基于桌面文档 `Oh-My-Codex（简称OMX）完整详解.md` 提炼；具体命令和 workflow 以当前安装版本的 catalog 为准。
