Oh-My-Codex 深度解析 - HTML PPT

打开方式：
1. 直接打开 index.html。
2. 或在本目录启动任意静态文件服务器后访问 index.html。

演示控制：
- 右方向键 / 空格 / PageDown：下一页
- 左方向键 / PageUp：上一页
- Home / End：第一页 / 最后一页
- F：进入或退出全屏
- N：显示或隐藏演讲者备注
- Esc：关闭演讲者备注
- 触控设备支持左右滑动翻页

内容结构：
- index.html：演示入口
- slides/：17 页独立 HTML 幻灯片
- shared/：共享样式和交互脚本
- assets/：从原 PPTX 提取的独立图片、图标和背景资源

说明：
- 页面不是整页截图。
- 文本框已转换为可选择、可搜索、可修改的 HTML 文本。
- 图形、连线和卡片由 CSS 元素绘制。
