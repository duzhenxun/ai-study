01. 工具定位：四大新增核心能力

持久化记忆系统 (MCP)

跨会话自动保存计划、执行状态与操作日志，任务意外中断后可无缝恢复，彻底解决AI的“健忘”痛点，大幅降低重复沟通成本。

专精角色 Agent 矩阵

内置33个高适配研发角色（架构师、测试工程师、运维专家等），沉淀37条标准化自动化技能，实现“专人专岗”的高效协作。

标准化研发闭环流水线

强制遵循规范流程：需求澄清 → 架构规划 → 并行执行 → 自测验证 → 修复闭环，确保每一步开发动作规范、可追溯且质量可控。

隔离式多AI并行协作网络

基于 Git Worktree + Tmux 实现开发环境隔离，多智能体可并行处理不同任务分支，避免代码冲突，提升研发吞吐效率。

核心执行基座 代码生成、文件读写、终端命令执行等硬核能力，统一由 Codex CLI 稳定支撑，保障执行层的健壮性。

中枢调度引擎 OMX 专注于任务智能调度、请求路由分发、全局状态管理与全流程的自动化编排，是系统的“大脑”。

多模型融合架构 支持 Codex、Claude、Gemini 等模型灵活混编，根据任务特性自动匹配最优模型，发挥各自技术优势。
